# Google Tag Manager (GTM)

[GTM]은 구글 마케팅 플랫폼에서 제공하는 태그 관리 서비스입니다.

데이터 분석에 필요한 다양한 조건과 스크립트들을 효율적으로 관리하고 공동작업 할 수 있습니다.
